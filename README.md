# Event Data Points Demo

A Crossref labs toy for plotting Events in 3D space. Just a bit of fun.

Connects to a MongoDB database maintained by event-data-query.

Config:

 - `QUERY_ELASTIC_URI`
 - `QUERY_DEPLOYMENT`
 - `QUERY_ELASTIC_URI`
 - `VIZ_3D_PORT`

To run:

    lein run
